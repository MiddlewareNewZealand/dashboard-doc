# Staff Sync Functions

### Relevant directories

`functions/ipayroll`

`functions/ipayroll/employees`

### Relevant files

#### function.sync.employees

Calls the `syncEmployees()` function from `service.employees.sync` at the given time periods.

#### service.sync.employees

Contains the builk of the business logic, particularly in `getStaff()`.

#### service.transform'

The `transformEmployeesResponse` function picks fields from the iPayroll API. Edit this to change fields synced to firestore.

:::tip
Here's a command to sync function edits to all staffsync functions

```console
firebase deploy --only functions:ipayrollStaffOnTopic, functions:ipayrollSyncStaffOnScheduleWeekdayFifteenMinute, functions:ipayrollSyncStaffOnScheduleWeekdayHour, functions:ipayrollSyncStaffOnScheduleWeekendHour 
```
:::
